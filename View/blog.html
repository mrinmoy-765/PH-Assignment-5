Important FAQ
What is Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface provided by web browsers that allows developers to interact with and manipulate the structure and content of a web page. When a web page is loaded, the browser converts the HTML into a hierarchical tree of objects, known as the DOM tree. Each element, attribute, and piece of text in the HTML document becomes a node in this tree. The DOM is essential for modern web development, as it provides the structure and programming interface that allows web pages to be interactive and responsive. Through JavaScript, the DOM enables developers to create dynamic and user-friendly websites.

How do you select an element from the DOM?

To select an element from the DOM, we can use several methods provided by the `document` object, each with its specific use case:
1. By ID: You can select an element using its unique `id` attribute.
2. By Class: You can select elements based on their `class` attribute.
3. By Tag Name: You can select elements by their HTML tag name.
4. By CSS Selector: You can use a CSS selector to select one or more elements based on more complex rules like class, ID, attributes, etc.
5. By Name Attribute: You can select elements with a specific `name` attribute, often used in forms.
These methods allow developers to target specific elements in the DOM for manipulation or interaction.

What is elevent delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM that allows you to handle events at a higher (parent) level rather than adding individual event listeners to each child element. Instead of attaching event listeners to multiple child elements, you attach a single event listener to their common ancestor. When an event occurs, it "bubbles up" to the ancestor, where it can be captured and handled.
Why Event Delegation is Useful:

  • Improves performance
  • Handles dynamic elements
  • Simpler code management

    • How do you manipulate an element attributes and styles using the DOM?

    You can manipulate an element's attributes and styles in the DOM using JavaScript. Here's how you can do both:
    1. Manipulating Attributes:
    we can get, set, or remove attributes of an element using DOM methods.

  • Getting an attribute: Use the getAttribute() method to retrieve the value of an attribute.
  • Setting an attribute:Use the setAttribute() method to add or update an attribute.
  • Removing an attribute:Use the removeAttribute() method to remove an attribute.

    • 2. Manipulating Styles:
    You can manipulate an element's CSS styles directly using the style property in JavaScript.
  • Inline style:Use the style property to modify an element's CSS properties directly.
  • CSS class manipulation:Use classList methods to add, remove, toggle, or check for a class.
